# Project Ideas

Список идей — **по одному проекту в день**: делаем, деплоим, ставим галочку.

Полный список в **[IDEAS.md](IDEAS.md)**.

## Как работать

1. Открываешь [IDEAS.md](IDEAS.md), выбираешь идею с ⬜
2. Создаёшь репо, делаешь проект
3. Деплоишь (Vercel / Railway / GitHub Pages)
4. В таблице ставишь ✅ и вставляешь ссылку на живой сервис
5. На следующий день — следующая идея
